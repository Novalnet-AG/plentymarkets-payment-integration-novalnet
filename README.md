# PLENTYMARKETS PAYMENT INTEGRATION BY NOVALNET
The Plentymarkets Payment Gateway by Novalnet enables secure integration of all local and international payments and access to a full-service payment offer from checkout to collection with simply one contract partners. It allows merchants to <a href= "https://www.novalnet.de/produkte"> automate their payment processing throughout the entire e-commerce value chain </a>, enrich customer experiences and improve conversion rates all at once.

## Integration requirements
The plugin is available for the 7.0.0 version in the following languages: EN & DE. Novalnet <a href= "https://www.novalnet.de/"> merchant account </a> is required for processing payments with Novalnet’s Plentymarkets Payment Gateway

## Key features 
* Easy configuration of 100 + payment methods (local + international) 
* Fully-automated fraud prevention & risk management including more than 60 modules
* One PCI DSS certified payment platform eliminating the need for own PCI certification

* Monitoring, reporting & easy payouts for all payment methods

## Integration steps
Follow these steps to successfully install the Plentymarkets Payment Integration by Novalnet:

1. Get the Plentymarkets Payment Plugin & detailed documentation <a href=  "https://marketplace.plentymarkets.com/en/plugins/payment/novalnet_5231"> from the marketplace. </a>
2. Install the module to your Plentymarkets shop
3. Configure merchant & payment details on your Plentymarkets shop
4. Let us know which payment types you wish to activate
5. You’re ready to go!
